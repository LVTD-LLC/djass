This doc is implementation-ready copy mapped to page sections for the single premium Djass offer.

## Audience and qualification intent

- Primary buyer: founders, solo builders, and product teams shipping multiple Django SaaS products or experiments.
- Desired buyer behavior: see $999 as a speed-and-repeatability decision, not a template purchase.
- Explicit disqualifier: low-intent template shoppers looking for the cheapest one-off download.

### 1) Hero badge

**One Plan. Premium Workflow.**

### 2) Hero headline

**$999 one-time for founders and teams shipping Django SaaS repeatedly**

### 3) Hero support text

Djass is for builders who want a faster, more repeatable way to start Django SaaS products. You get one premium plan with unlimited generations and lifetime updates.

If you only need a one-off template to tinker with, this is not the right product.

### 4) Pricing card title + price

- **Djass Premium Plan**
- **$999 one-time**

### 5) Core value bullets (ROI framing)

- Unlimited project generations for new SaaS products, experiments, and internal tools
- Lifetime updates to the starter so each new build starts from current best practices
- Faster project kickoff with a repeatable baseline you can trust
- Less setup drag from standardized scaffolding instead of ad-hoc project setup
- No recurring billing

### 6) Fit criteria section

**Who this is for**
- Solo founders and small teams building more than one Django product
- Product teams that want a predictable starting point for new apps and experiments
- Builders who care about shipping faster without inheriting a messy baseline

**Who this is not for**
- Template collectors looking for the cheapest one-off download
- Teams that prefer rebuilding infrastructure from scratch every project
- Buyers without an active Django SaaS roadmap

### 7) Proof/risk-reduction bridge

Want to inspect the foundation first? You can review the open-source baseline and self-host path before buying. Premium is for teams that want the hosted workflow and a faster way to start.

### 8) FAQ block (buyer objections)

**Why is this priced at $999?**  
Because it is built for people who repeatedly launch Django products. The value is in fewer setup cycles, fewer avoidable mistakes, and a more repeatable process every time you start something new.

**What exactly is included?**  
One premium plan: unlimited project generations, lifetime updates, and access to the same structured starter used to kick off Django SaaS products quickly.

**How do I know this is not just another template?**  
You can review the open-source baseline and self-host path before buying. Premium is for people who want the hosted workflow and a more repeatable way to start new products.

**Is this right for solo builders?**  
If you are shipping products regularly, yes. If you are only browsing for a cheap one-time template, no.
